## [해, 커리어] 다믐
다믐에서 사용하는 백엔드 개발 코드 보관소입니다.

### 활동 개요
- 요구 사항 분석 및 데이터베이스 설계
- 테스트 코드 작성
- 서비스에 필요한 API 개발

### 사용 기술
- spring boot starter (validation, data-jpa, web, security)
- jjwt
- lombok
- querydsl

### 데이터베이스
- h2 database

### 테스트
- Junit 4
- spring-restdocs-mockmvc
- spring-boot-starter-test
